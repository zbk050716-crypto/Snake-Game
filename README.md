# Snake Game

A classic Snake game developed using Unity and C#.

This project focuses on grid-based movement, game loop design, and simple system architecture.

## 🧩 Core Features

### 🕹 Gameplay System
- Grid-based movement system
- Direction control (Up / Down / Left / Right)
- Snake body following mechanism

### 🍎 Game Mechanics
- Random food spawning system
- Collision detection (wall & self-collision)
- Score tracking system

### 🔁 Game Loop
- Start / Play / Game Over loop
- Restart system after death

---

## 🧠 Technical Highlights

- Implemented grid-based movement system instead of free movement for deterministic gameplay
- Designed snake body follow logic using position queue/array tracking
- Built clean game loop state transitions (Start → Playing → Game Over)
- Separated gameplay logic for better readability and maintainability
